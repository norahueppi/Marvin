# Marvin Projekt

## Introduction

*Marvin is a attempt to recreate the moaning Doors from Hitchhikers Guide to the Galaxy*

## Features

- ESP32-A1S Audio Module
- Battery Powered
- SD-Card Audio Storage
- MP3 Capable
